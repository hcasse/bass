Connection

Before using Bass, you have to connect. Depending on the configuration, there are different ways:

With login
If you already a login, type login and password and find back your projects.
Anonymous
you may connect anonymously, usually for instant work or for testing, and creates your very first projects. Yet, you cannot later reconnect with this anonymous account.
Registration
you can register a new account: you have to provide at least a login and a password. The created account will be accessed later by classic logging.
Password forgotten?
With this option and if you have provided an email in registration option, you can reset your password and type a new one thanks to an email message.

Once connected, you can select a former project or create a new one:

In this dialog, you have a select a project to work with. Either you select a project on the left (and click Open), or you can create a new one on the right.

To create a new project, you have to type the name of the project (different from the existing ones), to select a template and to click on Create button. Templates provides different context of programming and experimenting: different architectures, different configurations of hardware and input/output devices. Each template has a name and a few lines of description.

Window Structure

Except for top-level bar with button, the screen is divided in 4 main panes:

  • left pane: display and configure register values (in debugging mode).
  • middle pane: display either the source to edir, or the disassembled code in debugging mode.
  • right pane: display the memory content or the state of sensors/actuators (both in debugging mode).
  • bottom pane: generic console displaying messages from the compilation, debugging, etc.

Source Edition

Depending on the chosen template, you may have one or several source files. For other ones, you can add as many sources as you want.

To edit a source, select it by clicking in the tab header and you get a classic editor:

This source editor supports classic keyboard shorts cut for fast edition with the tool button at top (from left to right):

  • Save: save the source in your project (on the server).
  • Undo: cancel the last modification./li>
  • Redo: redo the modification previously removed with Undo.
  • Cut: cut the selected text and store it in the clipboard.
  • Copy: copy the selected text in the clipboard.
  • Paste: paste at the current cursor position the content of the clipboard.
  • Search: search for a text in the source file.
  • Search and replace: search and replace a text in the source file.

User and Project Management

From the main screen, it also possible to manage the current user information and your project.

By clicking on the Project button, it is possible:

  • to open an existing project or create a new one,
  • to rename the current project,
  • to down load the sources of the current project,
  • or to delete the current project.

By clicking on the User button, you can:

  • configure the user (change name, email, password, etc),
  • or log out.

Notice that logging out an anonymous suer does not allow to log in to the same anonymous user later.
